# danitori 素材フォルダ

このフォルダに以下のファイル名で画像を置くと、LP に自動で表示されます。
ファイルが無い場合は、そのままプレースホルダーが表示されます（壊れません）。

| ファイル名        | 用途                                   |
|-------------------|----------------------------------------|
| `hero.jpg`        | ヒーローの商品ビジュアル                |
| `product.jpg`     | 商品紹介の使用イメージ（ナイトテーブル等）|
| `mat-product.jpg` | 使用できる素材：商品画像 サムネイル      |
| `mat-dani.jpg`    | 使用できる素材：ダニの写真や動画 サムネイル|
| `mat-video.jpg`   | 使用できる素材：使用動画 サムネイル      |
| `logo.png`        | favicon / OGP 用ロゴ                    |

## リンクの差し替え

`../index.html` 内 `CONFIG` を編集してください。

- `joinUrl` … 「この案件に参加する」
- `lineUrl` … LINE お問い合わせ
- `mailUrl` … メール問い合わせ
- `materials.{product,dani,logo,video}` … 各「選択する」ボタンのリンク先
