## まだ必要な画像（別途差し替え）

「使用できる素材」カードのサムネイル。置くと自動表示、無ければプレースホルダー。

| ファイル名        | 用途                          |
|-------------------|-------------------------------|
| `mat-product.jpg` | 使用できる素材：商品画像 サムネ |
| `mat-dani.jpg`    | 使用できる素材：ダニの写真や動画 サムネ |
| `mat-video.jpg`   | 使用できる素材：使用動画 サムネ |
| `logo.png`（任意）| favicon / OGP 用ロゴ           |

※「ロゴ素材」カードはテキスト表示のため画像不要。

## リンクの差し替え

`../index.html` 内 `CONFIG` を編集してください。

- `joinUrl` … 「この案件に参加する」
- `lineUrl` … LINE お問い合わせ
- `mailUrl` … メール問い合わせ
- `materials.{product,dani,logo,video}` … 各「選択する」ボタンのリンク先
